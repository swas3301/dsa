Solved: Merge Two Sorted Lists (LeetCode #21)

Today’s problem was a classic linked list question focused on merging two sorted lists efficiently.

🔹 Key Learning:
Since both lists are already sorted:

* Compare the current nodes of both lists
* Attach the smaller node to the merged list
* Move forward step by step

🔹 Approach:
Used an iterative approach with a dummy node:

* Maintain a `tail` pointer for the merged list
* Compare values from both lists and link the smaller one
* Attach the remaining nodes once one list ends

🔹 Complexity:
Time: O(n + m)
Space: O(1)

💡 This problem strengthened my understanding of linked list traversal and how dummy nodes simplify edge cases in list manipulation.
