Solved: Palindrome Linked List (LeetCode #234)

Today’s problem was a nice linked list challenge that combined traversal with palindrome checking.

🔹 Key Learning:
A linked list palindrome can be checked in different ways:

* Store values and compare from both ends
* Optimize later using fast/slow pointers + in-place reversal

🔹 Approach:
Used an array-based approach:

* Traverse the linked list and store all node values
* Compare values by popping from the end while traversing again
* If all values match → palindrome

🔹 Complexity:
Time: O(n)
Space: O(n)

💡 This problem reinforced that solving the problem correctly comes first, then optimizing space with advanced techniques later.
