Day 10 of my DSA Journey 🚀
Solved: Reverse Linked List (LeetCode #206)

Today’s problem was a fundamental linked list question—reversing a list. It’s simple in concept but super important for building strong pointer manipulation skills.

🔹 Key Learning:
Reversing a linked list is all about changing directions:

* Keep track of previous, current, and next nodes
* Reverse the links step-by-step

🔹 Approach:
Used an iterative approach:

* Traverse the list while updating pointers
* Reverse each node’s next pointer to point to the previous node

🔹 Complexity:
Time: O(n)
Space: O(1)

💡 This problem strengthened my understanding of pointer handling. Next step: solving it using recursion to deepen the concept further.
