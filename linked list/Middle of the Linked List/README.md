Solved: Middle of the Linked List (LeetCode #876)

Today’s problem was a simple yet important linked list question about finding the middle node efficiently.

🔹 Key Learning:
There are multiple ways to solve this problem:

* Store nodes in an array and directly access the middle
* Use the fast & slow pointer technique for better space optimization

🔹 Approach:
Used an array to store all nodes:

* Traverse the linked list and append nodes to a list
* Return the node at index `len(list) // 2`

🔹 Complexity:
Time: O(n)
Space: O(n)

💡 This problem reinforced that even simple problems can have multiple approaches, and comparing them helps build stronger problem-solving intuition. Next step: solving it using the fast & slow pointer approach.
