Solved: Remove Duplicates from Sorted List (LeetCode #83)

Today’s problem was a straightforward linked list question that emphasized in-place modification and careful traversal.

🔹 Key Learning:
Since the list is already sorted:

* Duplicate values will always be adjacent
* No extra data structure is needed
* We can remove duplicates by simply updating pointers

🔹 Approach:
Used an iterative approach:

* Traverse the linked list using a current pointer
* If the current node and next node have the same value, skip the duplicate node
* Otherwise, move to the next node

🔹 Complexity:
Time: O(n)
Space: O(1)

💡 This problem reinforced how problem constraints (like sorted order) can simplify the solution significantly.
