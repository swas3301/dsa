Day 19 of my DSA Journey 🚀
Solved: Remove Linked List Elements (LeetCode #203)

Today’s problem was a practical linked list manipulation problem focused on deleting nodes with a specific value.

🔹 Key Learning:
Removing nodes in a linked list requires careful pointer handling:

* Deleting the head node needs special handling
* A dummy node helps simplify edge cases
* Update links instead of deleting manually

🔹 Approach:
Used an iterative approach with a dummy node:

* Created a dummy node pointing to the head
* Traversed the list using `prev` and `curr` pointers
* If the current node matched the target value, skipped it by updating links

🔹 Complexity:
Time: O(n)
Space: O(1)

💡 This problem reinforced how dummy nodes make linked list operations cleaner and help avoid tricky edge cases.
