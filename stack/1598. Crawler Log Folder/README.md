Day 1 of my DSA Journey 🚀
Solved: Crawler Log Folder (LeetCode #1598)
Today’s problem focused on simulating folder navigation using simple logic. It’s a great exercise in understanding how to track state efficiently without overcomplicating things.
🔹 Key Learning:
Instead of actually building a file structure, I tracked the current depth:
"../" → move up one level (if possible)
"./" → stay in the same folder
"x/" → move into a child folder (increase depth)
🔹 Approach:
Used a counter to represent the current folder depth and updated it based on each operation.
🔹 Complexity:
Time: O(n)
Space: O(1)
💡 This problem reinforced the importance of simplifying problems by avoiding unnecessary data structures and focusing on what really matters.
Consistency is the key. Showing up every day, one problem at a time.
