Day 2 of my DSA Journey 🚀
Solved: Baseball Game (LeetCode #682)

Today’s problem was all about using a stack to efficiently simulate operations. It helped me understand how to manage dynamic data and apply operations based on previous states.

🔹 Key Learning:
Using a stack makes it easy to track and update scores:

* Integer → add new score
* "+" → sum of last two scores
* "D" → double the last score
* "C" → remove the last score

🔹 Approach:
Maintained a stack to store valid scores and updated it step-by-step based on each operation.

🔹 Complexity:
Time: O(n)
Space: O(n)

💡 This problem strengthened my understanding of stack-based problem solving and handling sequential operations efficiently.
