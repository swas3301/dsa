Day 3 of my DSA Journey 🚀
Solved: Valid Parentheses (LeetCode #20)

Today’s problem was a classic example of using a stack to validate patterns in a string. It really highlights how important order and pairing are in problem solving.

🔹 Key Learning:
A stack helps track opening brackets and ensures correct matching:

* Push opening brackets onto the stack
* When encountering a closing bracket, check for a valid match
* If mismatch or stack is empty → invalid

🔹 Approach:
Used a stack along with a hashmap to map closing brackets to their corresponding opening ones, making validation clean and efficient.

🔹 Complexity:
Time: O(n)
Space: O(n)

💡 This problem reinforced the concept of LIFO (Last In, First Out) and how stacks are perfect for handling nested structures.
