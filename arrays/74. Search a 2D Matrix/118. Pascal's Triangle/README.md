Solved: Pascal’s Triangle (LeetCode #118)

Today’s problem was a fun pattern-based problem that focused on building results incrementally.

🔹 Key Learning:
Pascal’s Triangle follows a simple rule:

* Every row starts and ends with `1`
* Each middle element is the sum of the two elements directly above it

🔹 Approach:
Built the triangle row by row:

* Started with the first row `[1]`
* Added padding (`0`) on both sides of the previous row
* Generated the next row by summing adjacent elements

🔹 Complexity:
Time: O(n²)
Space: O(n²)

💡 This problem reinforced how recognizing patterns can make implementation much simpler, especially in matrix/list construction problems.
