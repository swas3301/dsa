Day 29 of my DSA Journey 🚀
Solved: Same Tree (LeetCode #100)

Today’s problem was a nice tree comparison challenge that focused on checking both structure and node values.

🔹 Key Learning:
Two trees are considered the same only if:

* Their structures match exactly
* Corresponding nodes contain the same values
* Missing nodes (`None`) matter too

🔹 Approach:
Used recursive traversal:

* Traversed both trees in preorder
* Stored values (including `None` for missing nodes) in separate lists
* Compared both traversal results at the end

🔹 Complexity:
Time: O(n)
Space: O(n)

💡 This problem reinforced that when comparing tree structures, representing missing nodes is just as important as comparing actual values.
