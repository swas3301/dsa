Day 30 of my DSA Journey 🚀
Solved: Merge Two Binary Trees (LeetCode #617)

Today’s problem was a clean recursion-based tree problem that focused on combining two binary trees node by node.

🔹 Key Learning:
Tree merging becomes intuitive with recursion:

* If one node is missing, return the other
* If both nodes exist, combine their values
* Recursively merge left and right subtrees

🔹 Approach:
Used recursive DFS:

* Checked base cases when either tree node is `None`
* Added overlapping node values directly
* Recursively merged corresponding left and right children

🔹 Complexity:
Time: O(n) *(where n is the number of nodes visited)*
Space: O(h) *(recursion stack, where h is tree height)*

💡 This problem reinforced how naturally recursion fits tree problems, especially when the same logic applies repeatedly to subtrees.
