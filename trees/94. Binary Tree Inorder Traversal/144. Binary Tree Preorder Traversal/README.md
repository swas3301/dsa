Solved: Binary Tree Postorder Traversal (LeetCode #145)

Today’s problem helped me explore another important binary tree traversal pattern and deepen my understanding of tree recursion.

🔹 Key Learning:
Postorder traversal follows a specific order:

* Traverse the left subtree
* Traverse the right subtree
* Visit the current node

🔹 Approach:
Used recursive traversal:

* Recursively process the left child
* Recursively process the right child
* Append the current node’s value after both subtrees are processed

🔹 Complexity:
Time: O(n)
Space: O(n) *(due to recursion stack in worst case)*

💡 This problem reinforced how traversal order changes the outcome completely, and why understanding preorder, inorder, and postorder is essential for tree problems.
