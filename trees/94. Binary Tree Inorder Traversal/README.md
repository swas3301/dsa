Solved: Binary Tree Inorder Traversal (LeetCode #94)

Today’s problem was my introduction to binary tree traversal patterns. A simple problem, but an important step toward understanding tree-based data structures.

🔹 Key Learning:
Inorder traversal follows a specific pattern:

* Traverse the left subtree
* Visit the current node
* Traverse the right subtree

🔹 Approach:
Used recursive traversal:

* Recursively visit the left child
* Add the current node’s value to the result
* Recursively visit the right child

🔹 Complexity:
Time: O(n)
Space: O(n) *(due to recursion stack in worst case)*

💡 This problem reinforced the foundation of tree traversal and how recursion naturally fits hierarchical structures like trees.
